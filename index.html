"Build" instructions

I don't want there to be any chance of an executable file ending up on your server, so here's the command line:

ls ./16x16/ > icons.txt
sed -r "s/^([^\.]*)\.png$/.icon16.icon_\1 {background-image: url('16x16\/\1.png');}\n.icon32.icon_\1 {background-image: url('32x32\/\1.png');}/" icons.txt > icons.css
cat stub.css icons.css > farmfresh.css
rm icons.txt icons.css
